## install

```bash
git clone --bare git@github.com:yozul6/dots.git ~/.dotfiles
git --git-dir=$HOME/.dotfiles --work-tree=$HOME checkout --force
```

`--force` overwrites existing configs. back up first if needed.

## requirements

- **mangowm** (wayland compositor)
- **nerd font** (waybar icons, starship symbols)
- **foot** (terminal server + client)
- **fish** (shell)
- **swaybg** (wallpaper)
- **grim + slurp + wl-clipboard** (screenshot)

## configs
